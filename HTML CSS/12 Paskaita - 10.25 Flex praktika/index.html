CSS kintamieji

CSS'e kintamieji naudojami tam, kad nereiktu atsiminti specifiniu placiai naudojamu nustatymu svetaineje (spalva, tekstas ir t.t). Labai naudinga tureti kintamuosius norint globaliai pakeisti stilius daugeliui elementu.

CSS kintamuosius aprasome CSS (dazniausiai) root'o dalyje (:root{}) ir noredami kurti nauja kintamaji turime rasyti du bruksnelius ir kintamojo pavadinima, dvitaski ir reiksme (--kintamojo-pavadinimas: reiksme)

CSS kintamuosius naudojama kreipdamiesi i juos su var (variable) krepiniu ir skliausteliuosi nurodomu kintamojo pavadinimu (--kintamojo-pavadinimas)
