Temos kurias mokinsimes


HTML

Hyper text markup language - Hiper teksto zymejimo kalba skirta surasyti informacijai i puslapi ir ja grupuoti tagais


CSS

Cascading Style Sheets - Kaskadiniai stiliu sablonai skirti stilizuoti HTML


JS

JavaScript - skirtas prideti puslapiui funkcionalumo ir/ar reagavima i vartotojo ivestis


JS DOM

JavaScript Document Object Model - JavaScriptu keiciamas/redaguojamas/pildomas puslapio kontentas


TS

TypeScript - Lygiai tas pats kas JS, tik su griezta rasymo struktura, kurioje reikia nurodyti duomenu tipus


React

Galingas svetainiu kurimo irankis. Jo vienas didziausiu privalumu - niekada neda perkraunamas puslapis.


NodeJS

Node JavaScripte - skirtas kurti backEndui, kuriame pildomi ivairus veiksmai su duomenimis

Stilizavimo taga'i

  • i - Italic - pakreipia teksta
  • b - Bold - Pastorina teksta
  • em - Emphasize - pakreipia teksta ir ji pabreze (auto reader)
  • strong - Storina ir ji pabreze (auto reader)
  • sup -SubScript / super(man) - sumazina teksta ir pakelia i virsu
  • sub - SubScript - sumazina teksta ir pakelia i apacia
  • mark - pazymi teksto fona geltonai
  • u - underline - pabraukia zodi (bruksnys po zodziu)
  • big - padidina teksta
  • small - sumazina teksta
  • center - centruoja tekstine informacija
