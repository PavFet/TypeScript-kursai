Content

Elemento turinys tai ekrane atvaizduojama informacija vartotojui. Jinai gali buti tekstas, paveiksliukas, fonas... Turinio dydi nusako turinio kiekis / dydis (nustatytas per CSS) / display

Padding

Pamusalas yra atstumas tarp elemento turinio ir remeliu, kuris dazniausiai naudojamas tam, kad elemento turinys nebutu prilipes prie savo krastu.

CSS rasymo budai

  • padding: top right bottom left;
  • padding: top right/left bottom
  • padding: top/bottom right/left
  • padding: top/right/bottom/left;
  • padding-top: top;
  • padding-right: right;
  • padding-bottom: bottom;
  • padding-left: left;

Border

Remelis yra tarp Padding ir Margin. Remelis yra paskutine atvaizduojama elemento dalis (fonas). Kitaip pasakius - remelis yra matomo elemento krastas.

CSS rasymo budai

  • border: top right bottom left;
  • border-top: width type color;
  • border-right: width type color;
  • border-bottom: width type color;
  • border-left: width type color;
  • border-style: top /right /bottom /left;
  • border-width: top /right /bottom /left;
  • border-color: top /right /bottom /left;
  • border-right-color: color
  • border-radius: top /right /bottom /left; - apvalyna
  • border-[krastine]-[nustatymas]: nustatymo parinktis

Margin

Paraste yra atstumas nuo elemento remelio iki aplinkiniu elementu, kuris dazniausiai naudojamas tam, kad elementa patraukti nuo kitu elementu.

CSS rasymo budai

  • margin: top right bottom left;
  • margin: top right/left bottom
  • margin: top/bottom right/left
  • margin: top/right/bottom/left;
  • margin-top: top;
  • margin-right: right;
  • margin-bottom: bottom;
  • margin-left: left;
