Terminal commands

  • cd - change directory
  • ls - list files in current directory (terminal / gitBash / ...)
  • dir - list files in current directory (cmd)
  • mkdir - make directory - sukuria nauja aplankala
  • touch - sukurti nauja faila
  • cat - sukurti ir redaguoti nauja faila arba perziureti esama
  • clear - isvalo komandine eilute
  • rm - istrinti faila / aplankala (-r komanda trina rekursyviai)
  • CTRL C - atsaukia vykdoma veiksma
  • ARROW UP - automatiskai paleidzia praeitas paleistus comandas

Git setUp

  1. git config --global user.name "PavFet"
  2. git config --global user.email "fetiskinpavel@gmail.com (su kuriuo jungeisi pri github)"
  3. git config --global core.editor "code"
  4. git config --global core.autocrlf true (input jeigu linux/mac)
  5. git config --global -e (atidarys ir parodys info editoriaus)
  6. git config --help

Git commands

  • init - sukuria nauja gito repozitorija
  • status - parodo gito repozitorijos statusa
  • add - Prideda nurodyta faila/failus prie commitui paruostu failu (stage'ina)
  • commit - sukuria kodo versija (commit -m)
  • restore - Panaikina esamus pakitimus failuose / isima is commitui paruostu failu saraso (unstage'ina) jeigu yra prierasas --staged
