Transition

  • transition-property - nurodai kurie stiliai suteikiami vykstant tranzicijai
  • transition-duration - nurodai per kiek laiko ivyks tranzicija
  • transition-timing-function - nurodai kaip tranzicija vyksta per pateikta laika
  • transition-delay - nurodai po kiek laiko prades vykti tranzicija
  • transition - (transition-property - transition-duration - transition-timing-function - transition-delay)

Timing functions

  • linear
  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • step-start
  • step-end
  • cubic-bezier

Transform

Scale

Scale - scaleX; scaleY; scaleZ; scale; scale3d;
Scale nusako kaip keiciasi elemento dydis. Default dydis yra 1.

Translate

Translate: translateX; translateY; translateZ; translate; translate3d;
Translate nusako i kuria puse ir kokiu atstumu elementas pajudes.

Rotate

Rotate: rotateX; rotateY; rotateZ; rotate; rotate3d.
Rotate nusako kurioje asyje ir kiek elementas suksis.

Skew

Skew: skewX; skewY; skew;
Skew nusako kaip yra istempiami elemento kampai.

Animation

  • animation-duration - animacijos trukme
  • animation-timing-function - animacijos vykdymas per pateikta laika
  • animation-delay - animacijos uzlaikymas
  • animation-iteration-count - animacijos vykdymo kartai
  • animation-direction - animacijos vykdymo kriptis
  • animation-fill-mode - nusako kokia bus elemento busena pries ir/ar po animacijos
  • animation-play-state - animacijos busena
  • animation-name - animacijos vardas
  • animation - apjungia visus animacijos nustatymus

    • KeyFrames

    @keyframes animation-name { .... }
    Animacijos frame'ai aprasomi nuo (from) - iki (to) arba procentaliai nuo(0%) iki (100%).
    Animacijos laikas yra skirstomas frame'ams pagal ju procentalius nurodymus
