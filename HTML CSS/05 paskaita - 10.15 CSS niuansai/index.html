Paveldejimas

Is protevinio elemento, vaikiniai elementai paveldi tam tikrus stilius. Paveldimi stiliai dazniausiai buna kazkas paprasto kaip: teksto spalva, sriftas, dydis ir pns... Nepaveldimi yra dydzio, display, fono, remeliu ir pns. nustatymai.

Stiliu prioritetai

  1. Inline style (prie tago HTMLe parasius style="")
  2. ID
  3. Class
  4. Element Selector
  5. CSS order (in CSS code)
  6. Element default style
  7. Inheritence

Esant dviem selektoriam, kurie nusako to paties elemento vienodus stilius yra atsizvelgiama i tu selektoriu prioritetus. Auksciau esantys selektoriai, turi didesni prioriteta (ziureti i sarasa)
Esant dviem vienodo prioriteto selektoriams (keiciant vienoda stiliu) yra atsizvelgiama i CSS eiliskuman (kuris CSS parasytas veliau, tas ir bus taikomas)

JEIGU IMANOMA - NENAUDOTI !important prierasas prie stiliaus ignoruoja visus prioritetus ir suteikia ta stiliu ir niekas jo pakeist negali apart kito !important, kuris turetu aukstesni prioriteta.

Tagai gali tureti savitus stilius, kurie jiems suteikiami paciu narsykliu.
Kartais skirstingos narsykles turi minimalistiskai skirtingus numatytuosius stilius. Tai gali paveikti svetaines isvaizda.
Kartais numatytieji stiliai pagadina svetaines vaizda arba apsunkina programuotojo darba.

Yra keletas budu spresti nenorimu numatytu stiliu sukeliamas problemas.
Vienas is budu yra nuimti visus numatytuosius stilius nuo savo svetaineje naudojamu tagu: RESET CSS
Kitas (placiau naudojamas) budas yra normalizuoti numatytuosius stilius tarp skirtingu narsykliu ir dali ju panaikinti/pakeisti naudojant Normalize CSS.
